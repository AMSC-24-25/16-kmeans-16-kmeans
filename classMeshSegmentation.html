Detailed Description

template<class M>
class MeshSegmentation< M >

Performs segmentation (clustering) on a 3D mesh using K-Means.

This class applies a clustering algorithm to a given 3D mesh, grouping similar points based on a specified metric.

Template Parameters
MThe metric used for measuring distances between points on the mesh (EuclideanMetric, GeodesicDijkstraMetric, GeodesicHeatMetric).
