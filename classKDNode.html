Detailed Description

template<typename PT, std::size_t PD>
class KDNode< PT, PD >

Represents a node in a k-d tree.

This class defines a node structure used in a k-dimensional tree (k-d tree) for spatial partitioning. Each node contains a point, left and right children, and the split dimension.

Template Parameters
PTType of the point coordinates (e.g., float, double, etc.).
PDDimension of the data points.
