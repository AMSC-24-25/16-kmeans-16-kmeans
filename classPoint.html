Detailed Description

template<typename PT, std::size_t PD>
class Point< PT, PD >

Represents a mathematical point in a multi-dimensional space.

The Point class represents a point in an n-dimensional space, where n is defined by the template parameter PD. The class provides methods for basic vector operations like addition, subtraction, division, cross product (for 3D), and computing the norm. The point also holds an optional centroid (another Point object) and a unique identifier.

Template Parameters
PTThe type used for the coordinates (e.g., float, double).
PDThe number of dimensions the point has (e.g., 2 for 2D, 3 for 3D).
