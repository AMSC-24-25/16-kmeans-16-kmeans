Detailed Description

template<std::size_t PD>
class KDE< PD >

Implements Kernel Density Estimation (KDE) for centroid initialization.

This class derives from CentroidInitMethod and KDEBase to provide KDE-based centroid initialization for clustering algorithms.

Template Parameters
PDDimension of the data points.
