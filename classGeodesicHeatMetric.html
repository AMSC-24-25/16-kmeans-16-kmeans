Detailed Description

template<typename PT, std::size_t PD>
class GeodesicHeatMetric< PT, PD >

A class for computing geodesic distances using heat diffusion-based method.

This class inherits from the GeodesicDijkstraMetric class and provides an implementation of geodesic distances based on heat geodesics. It uses data from the igl library to compute these distances across a given mesh.

Template Parameters
PTType of the point (e.g., float, double)
PDDimension of the point (e.g., 3D)
