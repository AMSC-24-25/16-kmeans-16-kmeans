Detailed Description

template<typename PT, std::size_t PD>
class EuclideanMetric< PT, PD >

A class that calculates Euclidean distance for clustering and metric purposes.

This class provides functionality to compute Euclidean distances between points and can operate both on CPU and GPU. On the CPU, it uses a KDTree structure (with OpenMP) to efficiently find nearest neighbors and compute distances. On the GPU, it uses CUDA to parallelize the computation and speed up the process.

PTType of the point (e.g., float, double)
PDDimension of the point (e.g., 2D, 3D)
