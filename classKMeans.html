Detailed Description

template<typename PT, std::size_t PD, class M>
class KMeans< PT, PD, M >

Class implementing the K-Means clustering algorithm.

The class performs the K-Means algorithm by clustering a set of data points into a specified number of clusters. It supports various centroid initialization methods and utilizes a distance metric for clustering.

Template Parameters
PTThe type of the point coordinates (e.g., double, int).
PDThe number of dimensions of the points (e.g., 2 for 2D, 3 for 3D).
MThe type of the distance metric (e.g., EuclideanMetric, GeodesicMetric, GeodesicHeatMetric).

template<typename PT , std::size_t PD, class M >
void KMeans< PT, PD, M >::fit ( )

Fit method to perform the K-Means algorithm.

This method performs the K-Means clustering process:

  1. Initializes centroids,
  2. Assigns points to clusters based on the closest centroid,
  3. Recomputes centroids and repeats until convergence.
