Detailed Description

template<typename PT, std::size_t PD, class M>
class Kinit< PT, PD, M >

Abstract base class for determining the optimal number of clusters (k).

This class provides an interface for algorithms that determine the optimal number of clusters (k) in a dataset. It includes a pure virtual method findK() that must be implemented in derived classes.
