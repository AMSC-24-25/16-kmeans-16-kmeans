Detailed Description

template<typename PT, std::size_t PD, class M>
class SilhouetteMethod< PT, PD, M >

Implements the Silhouette method for determining the optimal number of clusters.

This class inherits from Kinit and uses the Silhouette score to evaluate cluster quality. It determines the optimal number of clusters by maximizing the Silhouette score.

Template Parameters
PTType of the points (e.g., float, double, etc.).
PDDimension of the data points.
MMetric used for distance calculation.
