Detailed Description

template<typename PT, std::size_t PD>
class RandomCentroidInit< PT, PD >

A class for initializing centroids randomly from the dataset.

This class implements the CentroidInitMethod interface and provides functionality to select centroids randomly from the given dataset. It also includes methods to calculate the centroid of a cluster.
