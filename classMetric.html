Detailed Description

template<typename PT, std::size_t PD>
class Metric< PT, PD >

Abstract class representing a generic metric for calculating distances.

This abstract class serves as a base for implementing various distance metrics, such as Euclidean or geodesic distances. It provides common functionality for setting up and working with centroids and points. Derived classes must implement the setup and fit_cpu (and optionally fit_gpu) methods.

Template Parameters
PTType of the point (e.g., float, double)
PDDimension of the point (e.g., 3D)
