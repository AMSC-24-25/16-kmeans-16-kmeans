Detailed Description

template<typename PT, std::size_t PD>
class KdNode< PT, PD >

Represents a node in a kd-tree, including its bounding box and child pointers.

A kd-tree (k-dimensional tree) is a space-partitioning data structure used for organizing points in a k-dimensional space. Each node represents a partitioning of space with a bounding box and may contain a single point (in leaf nodes).

Template Parameters
PTType of the point coordinates.
PDDimensionality of the points.
